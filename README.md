## About

The "Rain Date Designs" brand has been my own brand I created for my social media accounts with the
goal of starting to take commissions and make a personal image for myself. With this project, I wanted
to use the same imagery and add to my prexsisting brand. My branding reflects how I do my business and
my ideals. A rain date is a back up plan, it implies flexability.

## Process

Going into the assignment of making a digital buisness card, I wanted to keep my pre-established color 
scheme and I wanted to add an animated element. Instead of using Figma, I used Adobe Illustrator to make a
mockup of the layout and graphics of the site. I believe this made it easier in the long run for breaking 
down all of the individual elements to add them in HTML and CSS. For the animated rain graphic, I made the 3 
individual frames in Illustrator then I brought the frames into Photoshop and exported it as a gif file.
The fonts I used were "AkayaKanadaka" for the header-text and "Ice Cream" for the sub-headers. I wanted the 
fonts to be interesting but also readable.  I'm dyslexic so I wanted to fully make sure it was easily readble.
Because I have minimal text on my site, I had a little more freedom with how creative the fonts could be.
I also didn't want the site to be overwhelming to look at. Rain is associated with a calm feeling so I wanted
to keep it simple and somewhat predictable. No need to be unsure what's clickable and what's not.

## Challenges

One of the challenges with defining my own brand is making sure my brand is distinguishable and memorable. 
Starting with the name, I needed to come up with something no one else was already using. "Rain Date Designs"
was the first name that wasn't already taken. The bigger challenge was making the website itself. With this
being my first time making a website with coding languages and not just using a website builder, I struggled
with remembering and getting used to the coding process. I had a lot of assistence from classmates and the 
professor.
